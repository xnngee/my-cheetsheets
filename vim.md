# Vim --- Cheet Sheet

## Exiting
- `:q` \
*Закрыть Vim*
- `:qa` \
*Закрыть все файлы*
- `:q!` \
*Закрыть без сохранения*
- `:w` \
*Сохранить файл*
- `:wq` \
*Сохранить и закрыть*

## Editing
- `i` \
*Вставить текст в текущую позицию курсора (перед курсором)*
- `a` \
*Вставить текст после курсора*
- `Esc` [button] \
*Выйти из режима редактирования*
